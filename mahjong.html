MahJong Calculator

JavaScript Java OpenCV Drag & Drop REST API

Description du projet

MahJong Calculator est une application web permettant de reconnaître automatiquement les tuiles d'une main de MahJong à partir d’une photo, puis de calculer son score selon les règles officielles du jeu. Ce projet mêle vision par ordinateur, interface utilisateur interactive et logique métier complexe.

Le frontend est développé en JavaScript et offre une interface fluide avec drag & drop pour organiser les tuiles détectées ou en choisir manuellement. Le backend en Java gère la logique du jeu, le traitement des données, et communique avec une API REST.

Fonctionnalités principales

  • Upload d'une image contenant une main de MahJong
  • Détection automatique des tuiles avec OpenCV
  • Affichage visuel des tuiles reconnues
  • Éditeur manuel de la main avec glisser-déposer
  • Calcul détaillé du score et des combinaisons détectées
  • API REST pour le traitement des requêtes côté serveur

Défis et solutions

Le plus grand défi a été la reconnaissance d'image avec OpenCV, notamment l’identification précise de chaque tuile dans des conditions de lumière variées. Nous avons utilisé des techniques de traitement d'image et de matching de contours pour améliorer la robustesse du système.

Côté frontend, l’interface de sélection manuelle avec glisser-déposer a été conçue pour être intuitive et accessible, permettant à l’utilisateur de corriger ou compléter la détection automatique si nécessaire.
