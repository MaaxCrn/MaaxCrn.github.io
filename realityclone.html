Reality Clone

Flutter Unity Python TypeScript SQL API REST VR

Description du projet

Reality Clone est un projet de recherche en équipe de quatre, centré sur l’exploration de la technologie du Gaussian Splatting pour la reconstruction 3D réaliste à partir de photographies. Le projet repose sur un pipeline complet allant de la capture à la visualisation immersive en réalité virtuelle.

Il se compose de trois parties : une application mobile Flutter permettant la prise de photos, un serveur de traitement qui convertit ces images en modèles 3D via un algorithme de Gaussian Splatting implémenté en Python, et enfin, une application Unity permettant de visualiser les modèles reconstruits en VR.

Fonctionnalités principales

  • Application Flutter pour capturer une série d’images
  • Serveur TypeScript/Node.js avec API REST et base de données SQL
  • Traitement des images et génération des modèles 3D avec Gaussian Splatting en Python
  • Stockage et gestion des modèles générés
  • Visualisation en 3D et en VR dans une scène interactive via Unity
  • Pipeline modulaire facilitant l’ajout de nouvelles étapes de traitement

Défis et solutions

Le principal défi était d’intégrer des technologies très différentes dans un même projet. L’interopérabilité entre les modules Flutter, Python et Unity a nécessité une structuration claire du backend et une gestion rigoureuse des formats d’échange.

Un autre challenge résidait dans la compréhension et l’application du Gaussian Splatting, une technique récente de rendu 3D. Après une phase de recherche, nous avons réussi à l'adapter à notre pipeline de photogrammétrie pour produire des résultats exploitables en VR.
