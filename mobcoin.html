MobCoin

Android Kotlin SQL CI/CD ROOM API REST

Description du projet

MobCoin est une application mobile Android de suivi et de visualisation de données financières liées aux cryptomonnaies. Conçue en équipe de trois, elle permet aux utilisateurs de rechercher des cryptomonnaies, d'afficher leur évolution sous forme de courbes interactives, de consulter les plateformes d’échange disponibles, de gérer un portefeuille personnalisé, et de sauvegarder leurs coins favoris.

Le développement a suivi une méthodologie agile avec Trello pour la gestion des tâches, Figma pour la conception des maquettes, et Git pour le versioning. Le projet intègre également une base de données locale via Room (SQL) et un pipeline CI/CD pour automatiser les builds et les tests.

Fonctionnalités principales

  • Recherche de cryptomonnaies avec auto-complétion
  • Visualisation de courbes d’évolution des prix
  • Affichage des plateformes d’échange associées
  • Ajout et gestion des coins favoris
  • Création de compte utilisateur et authentification
  • Suivi d’un portefeuille personnalisé
  • Stockage local avec Room Database
  • CI/CD pour automatiser les builds et les tests

Défis et solutions

Un des défis majeurs a été la gestion de la synchronisation entre les données en ligne et la base de données locale. Nous avons mis en place un système hybride combinant les API REST et Room afin de garantir la fluidité et la cohérence des données, même hors connexion.

De plus, nous avons automatisé notre processus de développement grâce à un pipeline CI/CD qui vérifie et compile le code à chaque push, ce qui a permis d'assurer une qualité constante tout au long du projet.
